Kim bu xlsanal? Who is xlsanal?

Sunucu tarafında hayat bulan sistemler inşa etmekten zevk alıyorum. Node.js ekosistemi ve modern JavaScript ile backend API'lar, otomasyon araçları ve Discord botları geliştiriyorum.

Güvenlik konusuna özellikle önem veriyorum; log yönetimi, rate limiting, erişim kontrolü ve güvenli mimari tasarım benim işimin ayrılmaz parçaları.

Full-stack tarafında da rahatım — frontend'den backend'e, veritabanı tasarımından deployment'a kadar projenin tamamını sahiplenerek yürütebiliyorum.

I enjoy building systems that come alive on the server side. I develop backend APIs, automation tools, and Discord bots using the Node.js ecosystem and modern JavaScript.

Security is a core part of my workflow — log management, rate limiting, access control, and secure architectural design are essential parts of what I do.

I'm comfortable going full stack — from frontend to backend, database design to deployment, I can own an entire project end-to-end.
